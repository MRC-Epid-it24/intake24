Survey submissions #

Browse survey submissions #

Get list of survey submissions.

Request #

GET /api/admin/surveys/:surveyId/submissions
    ?search={searchText}
    &page={page}
    &limit={limit}

Authorization: Bearer {accessToken}
Content-Type: application/json

Response #

200 OK

{
    "data": [{...}],
    "meta": {...}
}
