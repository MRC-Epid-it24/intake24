As served sets #

Browse as served sets #

Browse paginated as served set list

Request #

GET /api/admin/images/as-served
    ?search={searchText}
    &page={page}
    &limit={limit}

Authorization: Bearer {accessToken}
Content-Type: application/json

Response #

200 OK

{
    "data": [
        {
            "id": string,
            "description": string,
            "imageUrl": string,
        },
        ...
    ],
    "meta": {...}
}

Create as served set #

Create new as served set entry

Request #

POST /api/admin/images/as-served

Authorization: Bearer {accessToken}
Content-Type: multipart/form-data

{
    "id": string,
    "description": string,
    "selectionImage": File
}

Response #

201 Created

{
    "id": string,
    "description": string,
    "selectionImageUrl": string,
    "images": [
        {
            "id": string,
            "weight": number,
            "mainImageUrl": string,
            "thumbnailUrl": string,
        },
        ...
    ],
}
