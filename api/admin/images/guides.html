Guide images #

Browse guide images #

Browse paginated guide image list

Request #

GET /api/admin/images/guides
    ?search={searchText}
    &page={page}
    &limit={limit}

Authorization: Bearer {accessToken}
Content-Type: application/json

Response #

200 OK

{
    "data": [
        {
            "id": string,
            "description": string,
            "imageUrl": string,
        },
        ...
    ],
    "meta": {...}
}

Create guide image #

Create new guide image entry

Request #

POST /api/admin/images/guides

Authorization: Bearer {accessToken}
Content-Type: application/json

{
    "id": string,
    "imageMapId": string,
    "description": string
}

Response #

201 Created

{
    "id": string,
    "description": string,
    "baseImageUrl": string,
    "imageMapId": string,
    "objects": [
        {
            "id": string,
            "description": string,
            "outlineCoordinates": number[],
            "weight": number
        },
        ...
    ],
}
