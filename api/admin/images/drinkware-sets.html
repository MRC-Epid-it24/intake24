Browse drinkware sets #

json
GET /api/admin/images/drinkware-sets
    ?search={searchText}
    &page={page}
    &limit={limit}

Authorization: Bearer {accessToken}
Content-Type: application/json

Response #

json
200 OK

{
    "data": [
        {
            "id": string,
