Schemes #

Browse schemes #

Browse paginated scheme list

Request #

GET /api/admin/schemes
    ?search={searchText}
    &page={page}
    &limit={limit}

Authorization: Bearer {accessToken}
Content-Type: application/json

Response #

200 OK

{
    "data": [{...}],
    "meta": {...}
}

Create scheme #

Create new scheme entry

Request #

POST /api/admin/schemes

Authorization: Bearer {accessToken}
Content-Type: application/json

{
    "id": string,
    "name": string,
    "type": 'data-driven',
    "meals": [{...}],
    "questions": {...},
    "export": [{...}]
}

Response #

201 Created

    ...
}
