# User profile

# Get profile data

Get logged-in user profile data

# Request

GET /api/admin/user

Authorization: Bearer {accessToken}
Content-Type: application/json

# Response

200 OK

{
    "profile": {
        "name": string,
        "email": string,
        "phone": string
    },
    "permissions": string[],
    "roles": string[]
}
