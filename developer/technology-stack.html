# Technology Stack

# Frontend

There are two frontend apps:

  • survey client
  • admin client

Both built on same tech stack:

  • Vue (opens new window) The Progressive JavaScript Framework
  • Vue Router (opens new window) - The official router for Vue.js
  • Vuex (opens new window) - The official router for Vue.js
  • Vue I18n (opens new window) - Internationalization
  • Vuetify (opens new window) - Material design based UI component framework
  • Axios (opens new window) - HTTP client

Vue version

Currently apps are built using Vue V2. Vue V3 is already released, but some of the 1st/3rd party plugins are still in process to be adopted for V3. Once Vue V3 plugins ecosystem catches up, we'll look into migration to V3.

# Backend

  • Express (opens new window) - HTTP server
  • Sequelize (opens new window) - database ORM
  • Sequelize-typescript (opens new window) - Decorators for Sequelize
  • Bullmq (opens new window) - queue system built on top of Redis

# Testing Framework

Both admin / survey front-ends and api-server backend are set up with Jest (opens new window) Testing Framework.

# Frontends

Frontends use Vue.js Jest implementation, so you can run tests with:

npm tun test

# Backend

Backend testing is set up for unit and integration tests. You can run tests with following commands.

Unit tests

npm run test:unit

Unit tests watch for development

npm run test:unit:watch

Integration tests

npm run test:integration

Integration tests watch for development

npm run test:integration:watch
