# Logging

Path: `packages/services/src/logger/config.ts`

## Log level
Log level to capture in logs

* object-path: `level`
* dotenv var: `LOG_LEVEL`
* type: `string`
* default: `'debug'`

## Log dir

Logs directory where all application logs are stored.

* object-path: `dir`
* dotenv var: `LOG_DIR`
* type: `string`
* default: `'storage/logs'`
