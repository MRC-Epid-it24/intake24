# References

## As served sets

Paginated `as served set` reference list

### Request

```json
GET /api/admin/references/as-served-sets
    ?search={searchText}
    &page={page}
    &limit={limit}

Authorization: Bearer {accessToken}
Content-Type: application/json
```

### Response

```json
200 OK

{
    "data": [{...}],
    "meta": {...}
}
```
